# Implementation Plan - Professional Portfolio V2

## Goal
Build a clean, multi-page personal portfolio emphasizing trust, calmness, and intelligence.

## Design System
- **Typography**: 'Inter', sans-serif (Clean, readable).
- **Colors**:
    - Light: Bg #F8FAFC, Text #1E293B, Accent #2DD4BF.
    - Dark: Bg #0F172A, Card #1E293B, Accent #38BDF8.
- **Layout**: Spacious centered container (max-width 1100px), consistent top nav.
- **Components**: Rounded cards (12px), soft shadows, subtle interactions.

## File Structure
- `index.html`: Home (Hero, CTA).
- `about.html`: Story-driven biography.
- `projects.html`: Project cards grid.
- `experience.html`: Professional timeline.
- `github.html`: Open source highlights.
- `contact.html`: Contact form.
- `style.css`: Design tokens, layout, responsiveness.
- `script.js`: Theme toggle, mobile menu, page transitions.
